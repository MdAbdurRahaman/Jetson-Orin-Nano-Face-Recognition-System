# Jetson Orin Nano Face Recognition

Real-time face recognition system using NVIDIA Jetson Orin Nano, OpenCV, and face_recognition.

## Features

- Real-time face recognition
- Face registration system
- Face encoding generation
- FPS display
- USB/CSI camera support
- Jetson optimized

## Run

### Register Face

python3 register_face.py

### Encode Faces

python3 encode_faces.py

### Start Recognition

python3 recognize_faces.py
